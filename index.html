About Me

단순히 텍스트가 아닌, 경험을 만듭니다.

화면 위에서 동작하는 버튼 하나, 최적화된 마크업 설계 하나가 사용자가 접하게 되는 웹 세상의 전부가 될 수 있다는 사실에 늘 매력을 느낍니다. 동양미래대학교에 입학한 이래로 소프트웨어의 기초 토대를 공부하며, 사용자 시선이 먼저 가 닿는 프론트엔드 분야를 연구하는 것에 매진하고 있습니다.

체계적으로 구조를 짜는 것을 좋아하며, 새로운 배움을 두려워하지 않고 한 줄의 코드를 작성하더라도 최적의 효율성을 고민하여 깔끔하게 마무리지으려 힘씁니다.

HTML5

의미론적(Semantic)이고 정돈된 구조적 마크업 설계를 수행하며, 최적화된 웹 접근성의 기초 뼈대를 설계할 수 있습니다.

CSS3

Flexbox 및 Grid 레이아웃을 능숙히 다루며, 미디어 쿼리를 기반으로 한 모바일 및 데스크탑 완벽 반응형 디자인 스타일을 구축합니다.

Git

버전 제어 시스템의 필수적인 기초 명령어를 학습하고, GitHub 원격 저장소 관리를 활용하여 포트폴리오 사이트를 안전하게 릴리즈합니다.

Python

기본적인 자료구조 및 흐름 제어, 데이터 정제 코드를 다룰 줄 알며 문제 해결을 위한 알고리즘적 설계를 유연하게 구성합니다.
